Quality Bike Store Backend 

key features:
* CRUD(create, read, update, delete) features available,
* user can order products and can see revenue,
* if user order a product with all quantiy then product can not be visible in products page.
* searching a product,


Technology used:
* Nodejs (runtime for javascript),
* ExpressJs( Web framework for nodejs),
* Mongodb database(store data),
* Mongoose ( for schema defination and data operation),  
* zod validation (no extra garbage data can be added),
* data stores in mongodb database,
